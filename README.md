# SkillThis Documentation

Documentation site for [SkillThis](https://skillthis.ai), the free tool that transforms your professional expertise into AI-ready skills.

**Live site:** [docs.skillthis.ai](https://docs.skillthis.ai)

## What is SkillThis?

[SkillThis](https://skillthis.ai) lets you describe what you do in plain text and generates a structured skill file that makes ChatGPT, Claude, or Cursor perform like a domain expert. No account required, no cost.
